# Digital Pandeji - Fractional CMO Website

This repository holds the code for my professional portfolio website: [digitalpandeji.com](https://digitalpandeji.com).

## To Update the Website:
1. Edit the `index.html` file directly in GitHub.
2. Commit the changes.
3. Wait 1-3 minutes for GitHub Pages to push the changes live.

## Form Integration
The contact form is integrated using a native HubSpot embedded form, which routes directly to the CRM.
